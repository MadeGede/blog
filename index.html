Multiplexer & Demultiplexer

Pemahaman Mendalam tentang Rangkaian Kombinasional dalam Teknik Digital

Teknik Digital merupakan fondasi penting dalam memahami sistem elektronika modern. Blog ini membahas dua komponen fundamental: Multiplexer (MUX) dan Demultiplexer (DEMUX), yang merupakan bagian dari rangkaian kombinasional dalam sistem digital.

Mengapa Mempelajari MUX & DEMUX?

Multiplexer dan Demultiplexer adalah komponen kunci dalam sistem digital karena:

  • Dasar Sistem Komunikasi: Untuk routing data dan seleksi sinyal
  • Arsitektur Komputer: Digunakan dalam CPU, memori, dan bus sistem
  • Optimasi Hardware: Mengurangi kompleksitas rangkaian
  • Aplikasi Luas: Dari sistem sederhana hingga kompleks

Pendahuluan Teknik Digital

Konsep Dasar Sistem Digital dan Rangkaian Logika

Sistem Bilangan Biner

Sistem digital bekerja dengan sinyal diskrit yang direpresentasikan dalam bentuk biner (0 dan 1). Sistem ini menjadi dasar semua operasi dalam teknik digital.

Tabel Konversi Biner ke Desimal

Biner (4-bit) Desimal Notasi
0000 0 I₀ / Y₀
0001 1 I₁ / Y₁
0010 2 I₂ / Y₂
0011 3 I₃ / Y₃
0100 4 I₄ / Y₄

Gerbang Logika Dasar

Dalam teknik digital, terdapat tiga operasi logika dasar yang membentuk semua rangkaian kompleks:

  • AND: Output 1 hanya jika semua input 1
  • OR: Output 1 jika minimal satu input 1
  • NOT: Membalik nilai input (0 menjadi 1, 1 menjadi 0)

Rangkaian Kombinasional

Rangkaian kombinasional adalah rangkaian digital yang outputnya hanya bergantung pada kombinasi input saat ini. Tidak seperti rangkaian sekuensial, rangkaian kombinasional tidak memiliki elemen memori.

Karakteristik Rangkaian Kombinasional

  • Tidak memiliki feedback atau elemen memori
  • Output = f(Input₁, Input₂, ..., Inputₙ)
  • Dapat direpresentasikan dengan tabel kebenaran
  • Dapat disederhanakan dengan peta Karnaugh
  • Contoh: Multiplexer, Demultiplexer, Decoder, Encoder

Diagram Blok Umum

Input₁ ──┐ Input₂ ──┤ ... ├── [Rangkaian Kombinasional] ── Output Inputₙ ──┘

Multiplexer (MUX)

Rangkaian Pemilih Data dalam Sistem Digital

Konsep Dasar Multiplexer

Multiplexer (MUX) adalah rangkaian kombinasional yang memilih satu dari beberapa input data dan meneruskannya ke output tunggal berdasarkan nilai seleksi.

Spesifikasi MUX

  • n input data (I₀, I₁, ..., Iₙ₋₁)
  • m bit seleksi (Sₘ₋₁, ..., S₀) dengan 2ᵐ = n
  • 1 output (Y)
  • Persamaan: m = log₂(n)

Tabel Kebenaran MUX 4:1

S₁ S₀ Output (Y) Input Terpilih
0 0 I₀ Input 0
0 1 I₁ Input 1
1 0 I₂ Input 2
1 1 I₃ Input 3

Demultiplexer (DEMUX)

Rangkaian Pembagi Data dalam Sistem Digital

Konsep Dasar Demultiplexer

Demultiplexer (DEMUX) melakukan fungsi kebalikan dari multiplexer. DEMUX mengambil satu input data dan mendistribusikannya ke salah satu dari banyak output berdasarkan nilai seleksi.

Spesifikasi DEMUX

  • 1 input data (I)
  • m bit seleksi (Sₘ₋₁, ..., S₀)
  • n output (Y₀, Y₁, ..., Yₙ₋₁) dengan 2ᵐ = n
  • Output tidak terpilih biasanya bernilai 0

Tabel Kebenaran DEMUX 1:4

S₁ S₀ Y₀ Y₁ Y₂ Y₃
0 0 I 0 0 0
0 1 0 I 0 0
1 0 0 0 I 0
1 1 0 0 0 I

Cara Menggunakan Simulator

Mode MUX: Klik pada input I₀-I₃ untuk mengubah nilainya (0/1). Ubah bit seleksi S₁ dan S₀ untuk memilih input mana yang akan muncul di output Y.

Mode DEMUX: Klik pada input untuk mengubah nilainya. Ubah bit seleksi untuk menentukan output mana yang akan menerima nilai input.
